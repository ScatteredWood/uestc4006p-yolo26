# UESTC4006P-YOLO26

This repository is based on the official Ultralytics YOLO framework and is adapted for the UESTC4006P Individual Project on road crack detection and segmentation.

## Overview

The project is used to explore YOLO26-based methods for pavement crack analysis, including:

- crack detection
- crack segmentation
- model comparison and improvement
- subsequent custom module integration such as CA neck

This repository is intended as the new experimental baseline for future development and validation.

## Current Focus

- building a clean YOLO26-based experimental branch
- reproducing official YOLO26 baselines
- adapting the framework to the road crack dataset
- preparing for further architectural improvements

## Notes
